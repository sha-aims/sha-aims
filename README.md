# Tony Shaska

Professor of Mathematics working in

- arithmetic geometry  
- algebraic curves and their moduli  
- computational algebra and explicit methods  
- machine learning and symbolic computation  
- neuro-symbolic and structured learning systems  

## Research Overview

My work focuses on the interaction between **algebra, geometry, and computation**, 
with recent emphasis on integrating **symbolic mathematical structures into machine learning**.  
I am particularly interested in developing architectures and methods that respect 
underlying algebraic and geometric properties of data.
